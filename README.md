# fish.io.ai
Krillin' it since 2017

To launch app make sure to have a conda env called "fishr" with the necessary dependencies.

```bash
# in dev
./fishr-cli.sh --dev --start
# in prod
./fishr-cli.sh --prod --start
```

To kill app:

```bash
# in dev
./fishr-cli.sh --dev --kill
# in prod
./fishr-cli.sh --prod --kill
```
